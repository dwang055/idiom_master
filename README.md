# Idiom Master

A clean, lightweight, single-file web application designed to help students effectively learn and retain English idioms. The app comes completely pre-loaded with **300 essential idioms** and provides an interactive, 100% distraction-free environment for vocabulary mastery.

## 🚀 Key Features

* **300 Pre-Loaded Idioms:** Ready to study instantly with definitions, real-world context sentences, and Chinese translations.
* **Interactive Flashcards:** Spaced-repetition friendly controls ("Need Review" / "Got It") with a dedicated "Starred" folder for targeting difficult words.
* **Match Game:** A fast-paced vocabulary mini-game to test recall speed by matching English idioms directly to their meanings.
* **Contextual Quiz:** Fill-in-the-blank style questions using authentic example sentences, complete with instant keyboard-shortcut navigation for a seamless experience.
* **Minimalist & Offline-Ready:** Built entirely inside a single standalone HTML file. Zero external dependencies, lightning-fast loading, and includes a native Dark Mode toggle.
